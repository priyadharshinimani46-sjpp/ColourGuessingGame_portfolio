# 🎨 Color Guessing Game

An attractive and interactive browser game where players try to guess the correct color name for a randomly displayed color box.  
Built entirely with **HTML, CSS, and JavaScript** — no frameworks required!

---

## 🚀 Features
- Randomly generated color box each round
- Three color options to choose from
- Instant feedback (✅ Correct / ❌ Wrong)
- Automatic new round after each guess
- Simple, responsive design
- Perfect for showcasing in a portfolio

---

## 🎯 How to Play
1. Open the game in your browser.
2. Look at the colored box displayed.
3. Choose the correct color name from the three options.
4. If you guess correctly, you’ll see **“Correct!”**  
   If not, you’ll see **“Wrong! Try again.”**
5. A new round starts automatically.
